README.TXT

Title:
　　　　Candy Rock Star: Unity-Chan Concert Project

Requirements:
　　　　Unity 4.5.x or higher

Note:
     You need to have Unity installed on a Mac or Windows PC 
     to make use of this project.

     To get the latest Unity installer, visit:
     http://unity3d.com/unity/download

How to use this project:
  1. Unzip the project’s ZIP files into a path that does not
     contain double-byte characters. 
  2. Open the project using your installed version of Unity.
  3. Open the “Assets/Scenes” folder in the Project window, and
     double-click on “Main.unity”.
  4. Play the scene using the Play button. 

Notes for Unity Free users:
     This project makes use of post effects and other features 
     that require Unity Pro. 

     To see these effects, consider using a trial of Unity Pro.
